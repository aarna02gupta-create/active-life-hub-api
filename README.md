
  # Sports and Fitness Discovery App
This app is built for anyone who wants to make fitness a fun, social, and accessible part of everyday life. Instead of endlessly searching for gyms, classes, or sports events, the Sports and Fitness Discovery App brings everything together in one place. Think of it as your personal guide to finding activities that match your vibe, your schedule, and your location
This is a code bundle for Sports and Fitness Discovery App. The original project is available at https://www.figma.com/design/HBpGBCnnMcqIEn01VCg1Yt/Sports-and-Fitness-Discovery-App.
Why I Built It
I wanted to solve a simple problem: staying active shouldn’t feel complicated. Whether you’re into yoga, running, football, or just looking for a local fitness event, this app helps you discover opportunities around you and connect with communities that share your interests.
Built With
Design: Figma
Frontend: React Native / Flutter
Backend: Node.js / Firebase
Database: Firestore / MongoDB
APIs: Location services + fitness activity APIs
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
